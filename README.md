--Emotional Movie Recommender Chatbot
Description

Ce projet consiste à créer un chatbot capable de détecter l'émotion d'un utilisateur à partir de son texte, en français et en anglais, et à recommander des films adaptés à son état émotionnel. Les recommandations sont basées sur un dataset TMDB enrichi avec les émotions associées à chaque film, en combinant les genres, les résumés et les slogans.

Le chatbot possède une interface graphique intuitive développée avec Tkinter, où l'utilisateur peut écrire ses messages, voir l'émotion détectée accompagnée d’un emoji, et obtenir une liste de films correspondants.

--Fonctionnalités

Détection d’émotions : joie, tristesse, colère, peur, réflexion, excitation, etc.

Recommandation de films basée sur les émotions.

Support bilingue : détection en français et en anglais.

Interface graphique conviviale avec chat et affichage des films recommandés.

Technologies

--Python 3.10+

Pandas pour la manipulation de données

Transformers (Hugging Face) pour la détection d’émotions

Tkinter pour l’interface graphique
